# 💪 MassivSelf — 30 Tage Selbstbewusstsein

> Dein persönlicher Begleiter für 30 Tage mehr Selbstbewusstsein.

MassivSelf ist eine iOS-App, die dich über 30 Tage mit täglichen Aufgaben, Affirmationen und Stimmungs-Tracking begleitet — Schritt für Schritt zu mehr Selbstvertrauen.

---

## Features

| Feature | Description |
|---|---|
| 🏠 **Home** | Tagesübersicht mit Daily Task, Affirmation und Mood Check-in |
| 📅 **Journey** | Dein 30-Tage-Fortschritt auf einen Blick |
| 📓 **Journal** | Täglich reflektieren und festhalten |
| 👤 **Profil** | Persönliche Einstellungen und Fortschrittsstatistiken |
| 🔔 **Benachrichtigungen** | Tägliche Erinnerungen damit du dranbleibst |
| ✨ **Onboarding** | Sanfter Einstieg mit Zielsetzung und Namenseingabe |

---

## Tech Stack

- **SwiftUI** — Komplett natives UI
- **SwiftData** — Lokale Datenpersistenz
- **iOS 17+**

---

## Requirements

- iOS 17.0 oder neuer
- Xcode 15.0+

---

## Build

```bash
git clone https://github.com/ixebit/MassivSelf.git
cd MassivSelf
open MassivSelf/MassivSelf.xcodeproj
```

In Xcode: Schema auswählen → Simulator oder Device → ▶ Run
